Some Important things you need to learn to become a Developer
Any one Programming Language!

You need to learn any one programming language in starting. You can go with C , C++ or pYthon or Java also. I suggest you to go with C and C++. because it easy to learn the all concept of programming.

HTML & CSS

HTML stands for HyperText Markup Language. It is used to design web pages using the markup language. Cascading Style Sheets is a style sheet language used for specifying the presentation and styling of a document written in a markup language such as HTML or XML.

Javascript

JavaScript (JS) is the most popular lightweight, interpreted compiled programming language. JavaScript, often abbreviated as JS, is a programming language and core technology of the Web, alongside HTML and CSS. 99% of websites use JavaScript on the client side for webpage behavior. Web browsers have a dedicated JavaScript engine that executes the client code.

Learn JS Frameworks and libraries.

A JavaScript framework is a set of JavaScript code libraries that provide pre-written code for everyday programming tasks to a web developer. Frameworks have a specific context and aid in creating web applications within that context. They provide you with a great deal of control over your app. for eg:- React,Vue,Express,Node Etc.

Databases.

Databases are used for websites to store and organize large amounts of data, which allows users to search and filter information, and web developers to retrieve data quickly Using Sql. Use API is the acronym for application programming interface — a software intermediary that allows two applications to talk to each other.

Last Step is Deployment or Hosting and Upload on GitHub

GitHub is a developer platform that allows developers to create, store, manage and share their code. It uses Git software, providing the distributed version control of Git plus access control, bug tracking, software feature requests, task management, continuous integration, and wikis for every project.
Websites are stored – or “hosted” – on a publicly-accessible computer (a server). Some websites require an entire server to themselves. Others can share a server with 100s of other websites. The storage space, and the features that come with it, make up your hosting plan.
